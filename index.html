Welcome to Sheets2Slides

The tool for allowing distraction free consumtion of your user stories

Here's how it works:

  1. Make a copy of this example sheet.
  2. Write your own user stories, but don't change the column names.
  3. Publish your sheet: File > Publish to web ... > Publish
  4. Grab the URL (see screenshot) and paste it into the field at the bottom of the page.
  5. Click on Go!
  6. Everytime you make changes to the sheet, just reload this page.
